# GitHub Importer

Issueやラベルを含めた完全なコピーを行うためのツールです。GitHub標準のインポートやフォークの場合ソースコード以外（IssueやLabel）が読み込まれませんがこのツールを使うことでそれらの一括インポートが可能になります。

## ユースケース

- Issueでタスクを設定したトレーニング用リポジトリの配布
- よく使うIssueやLabel、Boilerplateをまとめてスターターセットを作成
